To produce the Linux .deb installer:

Make sure the program has been compiled.

In a Terminal with administrator privilege, make sure have the path set to the correct version of Java.  (See setPath.sh)

Invoke BuildLinux.sh

Note:

You will need to change the following as appropriate:

In CreateLinuxApp.sh, set the --module-path.

In LinuxInstaller.sh, set the --app-version number and, if needed, copyright date.
